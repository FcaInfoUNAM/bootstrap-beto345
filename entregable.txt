¿Qué hace la clase bg-dark?
La clase bg-dark de Bootstrap aplica un fondo oscuro al elemento, utilizando el color definido en su paleta de colores predeterminada.

Color hexadecimal del fondo bg-dark:
#212529

¿Qué permite que la contraseña esté oculta en el formulario?
El atributo type="password" en el input.
Este tipo de input hace que los caracteres escritos se muestren como puntos o asteriscos para proteger la información del usuario.

¿Dónde deberías agregar CSS en el documento si modificas el estilo de alguna clase?
Se debe agregar en una etiqueta <style> dentro del <head> o en un archivo CSS externo enlazado después del archivo de Bootstrap para que pueda sobrescribir sus estilos.

¿Bootstrap incorpora algún mecanismo para realizar los cambios del paso 4?
Sí. Bootstrap incluye clases utilitarias como:

bg-* para fondos

text-* para colores de texto

bg-opacity-* para transparencia

btn-* para cambiar colores de botones

También se puede personalizar mediante variables Sass si se trabaja con Bootstrap compilado.
